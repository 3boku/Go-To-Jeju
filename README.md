# Go-To-Jeju
LLM과 함께하는 가사연애 도전기

# ⚙️ 개발 환경
- `Go 1.22.0`
- `HTML5 / CSS3 / Javascript`
- **IDE** : Goland 2024.1.1
- **Framework**
  - github.com/gin-gonic/gin v1.10.0
  - github.com/google/generative-ai-go v0.16.0
  - github.com/joho/godotenv v1.5.1
  - google.golang.org/api v0.188.0

# 사용방법
```shell
go get
go run app.go
```
후 HTML파일 실행
